# Bike Torque Specs

A lightweight offline PWA for looking up road bike fastener torque specifications on your iPhone (or any mobile browser).

**Data source:** Table E.1 from *Zinn & the Art of Road Bike Maintenance*, 4th Edition by Lennard Zinn (VeloPress, 2013).

## Features

- 236 torque specs across 14 categories
- Live search — results filter as you type
- Category filter bar
- Works 100% offline after first visit
- Add to iPhone home screen via Safari → Share → Add to Home Screen

## Usage

**Just visit the link in Safari on your iPhone — no account, no install, no app store.**

Once the page has loaded once, it works completely offline. To add it to your home screen:

Safari → Share button → Add to Home Screen → Torque → Add

That's it. Launches like an app and works with no network connection.
